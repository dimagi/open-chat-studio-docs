# Custom Action Testing

## Prerequisite 
Before testing, make sure the Custom Action is enabled for your chatbot — see [creating a Custom Action](../concepts/team/custom_actions.md)

## How to test
To test the Custom Action, you can open a chat with your Chatbot and type a message that triggers the action. The bot will make a request to the external service and return the response to you.

To see more detail about the request and response, you can enable [tracing](../concepts/tracing.md) in the *Advanced* tab of your Chatbot.
