API: Open Chat Studio export
Description: Read-only, team-scoped endpoints for the Open Chat Studio data sync/export. **Unversioned and intended only for OCS export** — it carries no backwards-compatibility guarantee and may change without notice.

TAG: Team

ENDPOINTS:

GET /api/export/auth_providers/
  Summary: Auth Providers
  Parameters:
    - cursor (query, string (optional)): Keyset pagination cursor returned as `cursor` by the previous page.
    - limit (query, integer (optional)): Maximum rows per page (default 100, capped at 1000).
  Responses:
    200:
      Content: application/json
      Schema: AuthProviderList
    400: Team has no registered public key; secret data cannot be sealed.
      Content: application/json
      Schema: SyncErrorDetail
  Security:
    apiKeyAuth
    tokenAuth
